## Usage

An avatar is a circular component used to wrap an image, icon or initials. They are often used to represent a contact or user. An image will take priority over inner text.
To create an avatar give a `div` or `span` tag a class name of `avatar`

| value    | details                                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------------------------- |
| `avatar` | Creates a circular element with a bg of 90% tint of the color set in the theme and font color as per theme variable |

## Size

There are three sizes for the avatar element (the default size is standard and does not need a classname)

| value   | details                                        |
| ------- | ---------------------------------------------- |
| `large` | Adds the large padding variables to the avatar |
| `small` | Adds the small padding variables to the avatar |

## Avatar stack

An avatar stack is used to represent a group or collection of people or contacts. It is a wrapper to go around 2 or more `avatar` components

| value          | details                                             |
| -------------- | --------------------------------------------------- |
| `avatar-stack` | A wrapper class to group and align avatars together |

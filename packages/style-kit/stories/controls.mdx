## Usage

Controls are html input elements often used in forms. The styles for the controls override the default input elements.

The three controls used are

1. Toggle
2. Checkbox
3. Radio

All three of the controls follow the same html format:

```html
<label class="<name-of-control">
  <input type="<type-of-input>" />
  <span class="<name-of-overriding-icon>"></span>
</label>
```

| Label            | Input      | Span        | Details                              |
| ---------------- | ---------- | ----------- | ------------------------------------ |
| `switch-input`   | `checkbox` | `slider`    | To create a toggle control           |
| `checkbox-input` | `checkbox` | `checkmark` | To create a checkbox style input     |
| `radio-button`   | `radio`    | `checkmark` | To create a radio button style input |

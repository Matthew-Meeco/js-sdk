## Type

There are four types of buttons in the Meeco style kit. To use just add a value below to the class name of the `button` tag.

| value       | details                                                          |
| ----------- | ---------------------------------------------------------------- |
| `primary`   | gives the button a bg of the theme's primary colour              |
| `secondary` | gives the button a bg that is 90% transparency of primary        |
| `text`      | removes bg of button and leaves just text                        |
| `icon-only` | creates a round button for an icon to be displayed in the middle |

## Size

There are two sizes for the buttons (except for `icon-only` where there is only a default size).

| value   | details                                                            |
| ------- | ------------------------------------------------------------------ |
| `large` | Adds the large padding variables & larger font size to the button  |
| `small` | Adds the small padding variables & smaller font size to the button |

## Options

The `text` button can handle an icon being included in the inner text along with text. See below for examples.

## Usage

Tags are a UI Component to represent a label or "tag" for another element such as a `card` or `tile`.
There are three types of tag classnames that you can add to a `span` tag

| value | details |
| --- | --- |
| `tag` | Classname for a basic span that has a 90% tint of tag font color set |
| `tag-edit` | Takes the `tag` styles and can handle an icon to the `right` of the inner text |
| `tag-icon` | Takes the `tag` styles and can handle an icon to the `left` of the inner text |

## Options

You can either use the Meeco icons or external icon fonts in the `<i>` tags

## CSS Options

The `tag` elements takes the variable `highlight` value for its background color and the variable `highlightText` value for its font color from the theme array. To change this create a custom-theme.
Else override the styles in your own stylesheet.

```css
.tag {
  background-color: grey;
  color: black;
}
```

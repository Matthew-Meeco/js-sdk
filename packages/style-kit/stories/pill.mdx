## Usage

Tabs are a UI component used for navigation.

A pill is a single "tab" that can be used outside of navigation

| value | details |
| --- | --- |
| `tabs` | A classname for a wrapper div to group the multiple single `tab` within it  |
| `tab` | A classname for a single element that sits within the `tabs` wrapper|
| `pill` | A single UI element used to add extra information |

## Options

Additional classnames to add to the `tab`

| value | details |
| --- | --- |
| `selected` | An additional classname to give styles to the single tab to show is it the selected option or active option  |


## CSS Options

The `tabs` wrapper element takes the `background` value from the `tabs` key in the theme array. To change this create a custom theme or override in own stylesheet.

The `tab` element takes the `foreground` value from the `tabs` key in the theme array. To change this create a custom theme or override in own stylesheet.

```css
.tabs {
  background-color: black;
},
.tab.selected {
  background-color: white;
}
```

## Usage

The field elements are custom styled HTML input elements. It is meant for input types of `text` `search` `date` `password` and `email`.

The label HTML elements are also custom styled.

No extra classnames are required.

### Dropdown element

A custom styled `select` element.

| value | details |
| --- | --- |
| `dropdown-input` | Overrides the default select html element styles |

## Custom css

`input` elements take up the width of their container.

To adjust the width of the element give the parent container a width value.

```css
.container {
  width: 100px;
  input {
    // has a width of 100% so will now be 100px
  }
}
```
